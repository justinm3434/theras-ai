# Theras AI — Website

Landing page for Theras AI, a next-generation hard-wired inference compute company.

## Stack
- Static HTML + Tailwind CSS (CDN)
- No build step required
- Deploy anywhere: Vercel, GitHub Pages, Netlify

## Quick Start
```bash
# Local preview — just open index.html in a browser
open index.html

# Or use any static server
npx serve .
```

## Deploy to Vercel
1. Connect this repo on [vercel.com](https://vercel.com)
2. Framework: Other
3. Output directory: `.` (root)
4. Deploy

## Deploy to GitHub Pages
1. Go to repo Settings → Pages
2. Source: Deploy from a branch → `main` / `/ (root)`
3. Save

## Structure
```
index.html          # Main landing page
public/images/      # Logo and brand assets

```
